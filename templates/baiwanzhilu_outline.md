# 「百万之路」实盘系列 — 可复用视频大纲模板

> **系列定位：** 公开账户 40万→100万，主策略：Sell Put / Sell Call / LEAPS / Synthetic Long + 定投底仓
> **更新频率：** 每周 1–3 期
> **参考标杆：** OptionsWithRyan（实盘数字直接冲击）× Stas Talks Stocks（宏观叙事深度）× 天哥风格（平实真实、个人视角）

---

## 视频结构总览

```
[HOOK]           账户数字直接冲击 — 引发好奇
[开场]           自我介绍 + 本期核心一句话

[第1部分]        大盘脉搏 — 发生了什么？为什么？
[第2部分]        仓位体检 — 这周赚了亏了多少？

[第4部分]        后市判断 — 大佬怎么说，我怎么看
[收尾]           Discord CTA → 【END】
```

预计时长：12–18 分钟

## 详细大纲

### 🎯 HOOK（0:00–0:40）— Ryan 风格，数字直接出

**核心原则：** 第一句话就是数字，制造情绪冲击，不解释，先引悬念。

```
模板A（账户上涨期）：
「上周账户涨了 $X，大盘同期跌了 Y%——来，我带你看看这钱是怎么来的。」

模板B（账户下跌期）：
「这周账户浮亏 $X，但我的现金流账户又进了 $Y——
  亏损和收租，能同时发生吗？今天讲清楚。」

模板C（市场剧烈波动期）：
「大盘这周上蹿下跳，[事件]，账户市值变化：$X。
  但我的操作只有一个字：[等/买/卖]。为什么？」
```

### 【第1部分】大盘脉搏（1:15–4:00）约 2.5 分钟

> **参考：Stas 风格** — 先给宏观背景，再谈仓位；让观众理解「在什么市场环境下」做了这些操作

#### 1A. 本期最重要的1–2件大事

**写作规范：**
- 只选最影响美股的事件，最多 2 个，宁少勿多
- 每个事件：是什么 → 为什么影响市场 → 数字佐证（SPX/VIX/板块涨跌）
- 用生活化类比解释机制

```
参考框架：
「这周市场的关键词只有一个：[主题词]。
 [事件简述]——这导致了[直接反应]。
 你可以把它理解成：[生活化类比]。
 SPX 这周[涨/跌] X%，VIX 从 Y 飙到 Z，说明[情绪解读]。」
```

**B-Roll：** SPX 日线图截图 / 新闻标题截图 / VIX 走势图

#### 1B. FedWatch 降息预期解读

```
「顺便看一下市场对美联储的预期——
 FedWatch 数据显示：[下次会议]降息概率 X%，不降息概率 Y%。
 跟上周比，市场[更鸽/更鹰]了，原因是[简述]。
 这对我们的期权仓位意味着：[一句话结论]。」
```

**B-Roll：** FedWatch 截图（天哥上传）

### 【第2部分】仓位体检（4:00–6:00）约 2 分钟

> **当天实时数据：** 发视频当天直接截图券商 App，展示当天盈亏，不回溯过去一周

#### 2A. 累计进度 + 当天盈亏

```
「快速看一下账户——目前总市值 $XX 万，距离 100 万的目标还差 XX%。」
「今天账户[涨了/跌了] $X，主要是 [标的A] 和 [标的B] 带动的。」
→ 自然过渡到第3部分的操作讲解
```

**场景变体：**

```
账户上涨日：
「弟兄们，今天美股大涨，账户增值了 $X。
 主要的增长来自 [标的A] 和 [标的B]——
 那 [标的A] 为什么涨这么猛呢？」

账户下跌日：
「说实话，今天账户浮亏 $X。但我的现金流账户又进了 $Y——
 浮亏和收租，能同时发生吗？今天讲清楚。」
```

**数据来源：** 券商 App 当天截图（当日盈亏页面），无需 CSV

**B-Roll：** 券商账户截图（当天净值变化页面）

#### 2B. 浮盈/浮亏主角（可选，60秒）

**规范：** 只在有值得说的变化时才讲，没有就跳过

```
浮盈主角（有趣时才讲）：
「[标的]今天涨了 X%，这是为什么？」
→ 直接导入第3部分的个股分析

浮亏主角（诚实局）：
「[标的]还趴着，账面浮亏 $Y。
 我的计划是[Roll/等待/加仓]，理由是……」
```

---

### 【第3部分】本期主角操作（6:00–14:00）约 7–8 分钟

> **核心原则（天哥 × Ryan 融合）：** 每期聚焦 1–2 个操作，讲深讲透，每个操作都是一个「教学时刻」
> **准备工具：** 使用 `/trade-analyzer` skill 上传交易截图，自动生成谈话要点和思路

**操作讲解框架（每个操作约 3–4 分钟，聊天式节奏）：**

```
① 我做了什么（10秒）
  「我在 [标的] 上 [操作类型]：
   [到期日] 到期，[行权价] 行权价，收到权利金 $X」

② 为什么在这个位置做（30秒）
  「选这个行权价的原因是 [Delta/技术位/心理位]。
   [标的]目前股价 $Y，这个行权价距离现价 Z%，
   我判断在到期日前跌到这里的概率很低，因为[理由]。」

③ 科普时刻（30秒，可选）
  「顺便解释一下 [期权概念/策略逻辑]——
   用一个类比：[生活化类比]
   所以本质上，我做的事情就是[通俗化结论]。」

④ 风险在哪 + 如何处理（20秒）
  「当然，风险是存在的：如果 [标的] 跌破 [行权价]，
   我会选择 [Roll 展期/接受行权/止损]，
   因为[理由]。」
```

**B-Roll 配置：**
- OptionStrat 截图：行权价、到期日、盈亏曲线
- Fidelity 历史交易截图：实际成交记录
- PPT：操作前后对比 / 策略参数可视化
- Remotion 动画（仅当解释多步骤流程时）

---

### 【第4部分】后市判断（12:00–15:00）约 3 分钟

> **核心：** 用 market-intelligence-briefing skill 自动聚合大佬观点，天哥加个人判断层

#### 4A. 大佬观点速览（90秒）

```
「来看看最近各路大佬怎么说——」

必选来源（按照 market-intelligence-briefing skill 输出）：
• Tom Lee / Fundstrat：[最新观点 + 目标价/时间框架]
• Dan Ives / Wedbush：[科技股看法]
• 美联储：[最新表态]
• 白宫/关税：[政策信号]
• 选填：Elon Musk / 其他

输出格式：每人一句话，附上来源和日期，不做价值判断。
```

**B-Roll：** PPT — 观点汇总表（来源 | 观点 | 日期）

#### 4B. 天哥补充两句（60秒）

> **原则：** 尊重大佬观点，不打分不评判，天哥在讲完后以个人角度补充

```
「以上是各路大佬的看法。我补充两句——
 [个人观察/补充角度/对自己仓位的影响]。
 基于这些，我接下来的操作方向是[保守/激进/维持]。」

风险提示：
「当然，我也可能是错的。
 如果[反向情景]发生，我会[应对方案]。」
```

## 可变模块说明

根据当期市场情况，以下模块可以「扩展」或「压缩」：

| 模块 | 平静市（压缩） | 波动市（扩展） |
|------|--------------|--------------|
| 第1部分 大盘分析 | 1.5 分钟 | 4 分钟（重大事件时） |
| 第2部分 仓位体检 | 2 分钟（无特别变化） | 4 分钟（仓位大变动时） |
| 第3部分 主角操作 | 1 个操作，3 分钟 | 3 个操作，7 分钟 |
| 第4部分 后市判断 | 2 分钟（大佬观点一致） | 4 分钟（分歧大，天哥重点说判断） |
