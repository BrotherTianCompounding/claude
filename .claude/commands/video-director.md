# 视频导演标注系统

## Overview

接收已完成的文案脚本，输出两样东西：
1. **完整标注版脚本** — 标明每段画面使用什么 B-ROLL 素材（HOOK 和片尾除外）
2. **设计文档** — 针对需要制作的 PPT 页面，给出详细设计说明供美工执行

---

## 画面结构规则

本频道采用 **B-ROLL 为主、A-ROLL PIP 为辅** 的画面结构：

- **HOOK（开场）和片尾（END）**：全屏 A-ROLL（主播出镜），不叠加 B-ROLL
- **中间正文部分**：画面始终是 B-ROLL，右下角保留主播小头像（Picture-in-Picture）

因此，导演的核心工作是为中间正文部分编排 B-ROLL：决定每段台词对应什么画面素材。

---

## 可用 B-ROLL 素材类型

| 类型 | 代号 | 适用场景 |
|------|------|----------|
| PPT页面 | `B-ROLL PPT` | 概念解释、数据对比、关键结论、要点归纳、流程图、比喻可视化、仓位配比、公式推演 |
| Moomoo K线图 | `B-ROLL Moomoo K线图` | 股价走势、技术面分析、均线/RSI等指标展示、K线形态 |
| OptionStrat截图 | `B-ROLL OptionStrat` | 期权策略图、盈亏曲线、Greeks数据、期权链 |
| 券商期权链 | `B-ROLL 券商期权链` | 展示真实的期权链界面、行权价选择、权利金报价 |
| 历史交易截图 | `B-ROLL 历史交易截图` | 展示实际持仓、账单、盈亏记录、Discord操作截图 |

## 标注格式规范

```
[A-ROLL HOOK 开始]
……HOOK台词（全屏出镜）……
[A-ROLL HOOK 结束]

[B-ROLL PPT 三阶段路线图 开始]
……台词内容（画面是PPT，右下角PIP主播）……
[B-ROLL PPT 三阶段路线图 结束]

[B-ROLL Moomoo K线图 QQQ日线200SMA+RSI 开始]
……台词内容……
[B-ROLL Moomoo K线图 QQQ日线200SMA+RSI 结束]

[B-ROLL OptionStrat TQQQ $45 Put 32DTE盈亏曲线 开始]
……台词内容……
[B-ROLL OptionStrat TQQQ $45 Put 32DTE盈亏曲线 结束]

[A-ROLL END 开始]
……片尾台词（全屏出镜）……
[A-ROLL END 结束]
```

## B-ROLL 素材选择规则

- 讲解**当前真实存在**的期权策略参数、展示行权价/到期日/权利金/盈亏曲线 → **OptionStrat截图** 或 **券商期权链**
- 讲解股价走势、技术面分析、均线/RSI → **Moomoo K线图**
- 展示实际持仓、历史账单、Discord操作 → **历史交易截图**
- 解释概念、数据对比、计算推演、流程图、总结要点、仓位配比 → **PPT页面**
- 当一段台词没有明确的截图/K线对应时，默认使用 **PPT页面**

**关键规则：OptionStrat / 券商期权链只能展示当前真实数据，无法展示未来或假设场景。** 凡是脚本中出现"假设股价跌到XX""如果被行权之后卖Call"等假设性、未来性的场景，一律使用 **PPT页面** 展示（自行设计画面），绝不标注为 OptionStrat 或券商期权链。只有当前可以直接截图的真实数据才用 OptionStrat。

## 设计文档格式

对每一个需要制作的 PPT 页面，在脚本标注完成后，附上对应的设计文档：

```markdown
---
### 设计文档：[素材名称]
**类型：** PPT页面
**对应脚本位置：** 第X部分，台词「……」处
**时长：** 约X秒

**画面内容：**
[具体描述画面里有什么元素、文字、数字、布局]

**配色/风格：**
[主色调、字体风格、是否与频道VI一致]

**文案内容（画面上显示的文字）：**
[直接给出画面上要出现的所有文字]
---
```

对于 OptionStrat截图、Moomoo K线图、券商期权链、历史交易截图，无需设计文档，只需在标注标签中写清楚展示什么内容即可（天哥自行截图）。

## 录制版排版规范

标注版脚本完成后，额外输出一份「录制版脚本」，规则如下：

1. **逐句断行**：每句话独占一行，以句号、问号、感叹号、省略号为断句依据。方便主播逐句录制、逐句对照。
2. **结束标签后断两行**：每个标签结束后面，必须空出**两个空行**（即视觉上有一整段空白），明确区分不同镜头段落。
3. **保留所有标签**：开始/结束标签原样保留，不做任何删减。

---

## 标注行为规范

**必须做：**
- 每个标注块都有明确的开始和结束标签
- B-ROLL 标签里写清楚「展示什么」，不要只写类型
- 需要设计的PPT素材，每一个都出设计文档，不能遗漏

**不要做：**
- 中间正文部分不要标 A-ROLL（全部是 B-ROLL + PIP）
- 不要设计过于复杂的PPT页面，以实用清晰为主
- 设计文档里的文案直接可用，不要留"待填写"占位符
