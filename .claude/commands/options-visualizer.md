# 期权操作可视化文案生成器

## Overview

输入一笔期权操作的参数，自动生成：
1. **「第3部分主角操作」完整台词** — 按天哥的四步框架写好，直接录制
2. **B-Roll 标注** — OptionStrat 截图、历史交易截图、PPT 的具体标注
3. **PPT 设计说明** — 本次操作的参数可视化页面设计文档

---

## 输入格式

天哥提供以下信息（支持任意格式，能读懂即可）：

```
必填：
• 标的：如 NVDA、QQQ、COHR
• 策略类型：Sell Put / Sell Call / LEAPS Call / Synthetic Long / PMCC / Covered Call
• 行权价：如 $800
• 到期日：如 2026-06-20
• 收到权利金 / 花费成本：如 收到 $1.85/share，共 $185

选填（提供了会让台词更丰富）：
• 当时股价：如 $875
• 选择这个行权价的理由：如「技术支撑位」「Delta 0.2」「心理位」
• 有没有科普点：如「想解释一下什么是 PMCC」
• 风险管理计划：如「跌破就 Roll 到下个月」
```

---

## 四步台词框架（必须遵守）

每笔操作按以下四步生成台词，顺序不变：

```
① 我做了什么（10秒）
   直接报参数：标的 + 策略 + 行权价 + 到期日 + 权利金
   第一次提到标的要用全称，如「英伟达（NVDA）」

② 为什么在这个位置做（30秒）
   解释行权价选择逻辑：Delta / 技术位 / 距现价百分比
   必须有一个「概率感」的表述：「我判断在到期前跌到这里的概率很低」

③ 科普时刻（30秒，可选）
   如果这个操作适合科普一个概念，用生活化类比解释
   常见科普点：
   - Sell Put = 「收保证金」「卖保险」
   - LEAPS = 「用小钱控制大仓位」「以小博大的工具」
   - PMCC = 「让LEAPS帮你收租」「仓位自我造血」
   - Roll 展期 = 「把到期日往后推，同时换行权价」
   - Synthetic Long = 「用期权模拟持股，但只需要一小部分保证金」

④ 风险在哪 + 如何处理（20秒）
   诚实说风险，不回避
   给出具体的应对方案（Roll / 接受行权 / 止损标准）
```

---

## 语气规范

- **平实 > 专业**：说「我选这个价位是因为……」不说「根据Delta中性理论……」
- **真实 > 权威**：收了多少就说多少，不夸大，未到期的说「在途」
- **类比必须生活化**：期权概念用日常场景打比方，不用学术语言
- **标的全称**：第一次出现用「英伟达（NVDA）」，后续可简称

---

## 策略类型对应的科普重点

| 策略 | 核心类比 | 常用科普切入点 |
|------|---------|-------------|
| Sell Put | 「收保证金让别人把股票卖给我」/ 「卖保险」 | 为什么要在支撑位卖？被行权了怎么办？ |
| Sell Call / Covered Call | 「收租金」/ 「让手里的股票帮我打工」 | 什么时候适合卖？封顶收益怎么理解？ |
| LEAPS Call | 「用杠杆控制大仓位」/ 「期权里的长期债券」 | 为什么买LEAPS不买正股？Theta怎么影响LEAPS？ |
| Synthetic Long | 「用期权模拟持股」/ 「期权组合拳」 | 买Call+卖Put怎么构成合成多头？需要多少保证金？ |
| PMCC | 「让LEAPS收租」/ 「仓位自我造血」 | 什么是穷人版Covered Call？Roll的时机是什么？ |
| Roll 展期 | 「把债往后推，顺便换个利率」 | 什么时候该Roll？Roll的成本怎么算？ |

## B-Roll 标注规范

台词生成后，自动标注每段话对应的 B-Roll，格式与 video-director 一致：

```
[A-ROLL X 开始]
……台词……
[A-ROLL X 结束]

[B-ROLL OptionStrat {标的} {行权价}{策略} 盈亏曲线 开始]
……台词（嘴里说参数时）……
[B-ROLL OptionStrat {标的} {行权价}{策略} 盈亏曲线 结束]

[B-ROLL 历史交易截图 {标的}{策略}成交记录 开始]
……台词（说「大家看这张成交记录」时）……
[B-ROLL 历史交易截图 {标的}{策略}成交记录 结束]
```

**A-ROLL vs B-ROLL 判断：**
- 说「我做了什么」时 → 立刻切 OptionStrat 截图
- 说「大家看这张成交记录」时 → 历史交易截图
- 解释概念、数据对比 → PPT
- 解释 Roll 展期、合成多头构建等多步骤流程 → Remotion 动画（需单独说明）

## PPT 设计文档（每次必出）

每笔操作自动生成一份操作参数 PPT 设计说明：

```markdown
---
### 设计文档：[标的][策略] 操作参数页
**类型：** PPT页面
**对应脚本位置：** 第3部分，台词「我在……上……」处
**时长：** 静态展示约 15–20 秒

**画面内容：**
左侧大字：策略名称（如「Sell Put」）
中间参数表：
  标的：[全称]（[代码]）
  行权价：$[X]
  到期日：[日期]（距今约 X 天）
  权利金：$[X]（每股 $X）
  当时股价：$[X]
  行权价距现价：X%

右侧小字：「如果到期前不跌破 $X，这笔权利金全部落袋」

**配色：**
- 背景：深色（#1a1a2e 或品牌深蓝）
- 参数数字：白色加粗
- 权利金金额：绿色高亮
- 行权价：黄色高亮
---
```

```
## 四、Remotion 动画说明（如需要）

```
