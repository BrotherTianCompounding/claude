## 执行步骤

### 第一步：识别操作参数

从截图中提取以下信息（提取不到的标注「未识别」）：

- 标的：如 TSLA、NVDA、QQQ
- 操作方向：Buy to Open / Sell to Open / Buy to Close / Sell to Close
- 策略类型：Sell Put / Sell Call / LEAPS Call / Synthetic Long / PMCC / Covered Call / 其他
- 合约数量
- 行权价
- 到期日
- 成交价/权利金
- 订单类型：Day / GTC

**组合策略识别：**
| 截图特征 | 策略判断 |
|---------|---------|
| Sell to Open + Put | Sell Put（卖看跌期权） |
| Sell to Open + Call（有持股） | Covered Call（备兑看涨） |
| Buy to Open + Call（到期 > 1年） | LEAPS Call |
| Buy Call + Sell Put，同行权价同到期日 | Synthetic Long（合成多头） |
| Buy LEAPS Call + Sell 近期 Call | PMCC（穷人版备兑） |

如果截图包含多笔操作构成组合策略（如合成多头 = Buy Call + Sell Put），自动识别并计算净成本/净收入。

**提取完成后，先列出参数让天哥确认，再继续分析。**

### 第二步：搜索补充数据

用 WebSearch 获取：
1. 当前股价 — 计算行权价距现价的百分比

3. 该标的近期走势 — 辅助判断操作方向的合理性

## 可以聊什么？

### 1. 这笔操作的核心逻辑
- 为什么选 [标的]？（结合近期新闻/走势）
- 为什么用 [策略] 而不是直接买股票？
- 这个行权价 / 到期日的选择逻辑是什么？
- 距现价 X%，意味着……

### 2. 可以科普的概念
- [策略名称] 是什么？一两句话概括
- 可以提：「不理解的朋友可以参考我之前关于 [概念] 的视频」
- 生活化类比建议

### 3. 数字亮点（让观众记住的关键数字）
- 净成本 / 净收入
- 盈亏平衡点
- 最大风险 / 最大收益
- 年化回报率（如适用）
- 与直接买股票的资金对比

### 4. 风险在哪？怎么处理？
- 最坏情况
- 应对方案建议：Roll 展期 / 接受行权 / 止损
- 保证金 / Margin 提醒（如适用）

### 5. 和百万之路系列的关系（可选）
- 这笔操作对账户整体的影响
- 和当前大盘环境的关系

---

## 规则

- **只给思路，不写台词** — 天哥自己发挥
- **数字必须具体** — 不说"成本较低"，说"净成本 $3,000，相当于直接买股票的 X%"
- **必须包含风险点** — 每笔操作都要有风险提示
- **科普建议要实用** — 只推荐真正值得科普的概念，不是每笔都需要
- **提醒关联视频** — 如果某个概念之前视频讲过，建议天哥提一句
