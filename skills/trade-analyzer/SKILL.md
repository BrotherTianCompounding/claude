# 交易截图分析器 — 谈话要点生成

## Overview

天哥上传交易截图后，自动完成：
1. **识别操作参数** — 从截图中提取关键信息
2. **生成谈话要点** — bullet point 思路清单，不写台词
3. **搜索当前股价** — 补充实时数据，增强分析

**定位：** 这不是脚本生成器。天哥在镜头前用聊天式风格自由发挥，这个 skill 只负责帮他理清思路、确保不遗漏关键点。

---

## 第一步：识别操作参数

从截图中提取以下信息（提取不到的标注「未识别」）：

```
• 标的：如 TSLA、NVDA、QQQ
• 操作方向：Buy to Open / Sell to Open / Buy to Close / Sell to Close
• 策略类型：Sell Put / Sell Call / LEAPS Call / Synthetic Long / PMCC / Covered Call / 其他
• 合约数量：如 1 Contract
• 行权价：如 $360
• 到期日：如 Jan 21 2028
• 成交价/权利金：如 $111.99（买入）/ $81.99（卖出）
• 订单类型：Day / GTC
```

如果截图包含多笔操作（如合成多头 = Buy Call + Sell Put），自动识别为组合策略。

**确认步骤：** 提取完成后，先列出参数让天哥确认，再继续分析。

---

## 第二步：搜索补充数据

用 WebSearch 获取：
1. **当前股价** — 计算行权价距现价的百分比
2. **近期该标的新闻**（1-2条）— 辅助解释「为什么选这个标的」
3. **该标的近期走势** — 辅助判断操作方向的合理性

---

## 第三步：生成谈话要点

输出格式为 **bullet point 思路清单**，分以下几个板块：

### 输出模板

```markdown
# [标的] [策略类型] — 谈话要点

## 操作参数速览
| 项目 | 内容 |
|------|------|
| 标的 | [全称]（[代码]） |
| 策略 | [策略类型] |
| 行权价 | $[X] |
| 到期日 | [日期]（距今约 X 天） |
| 权利金/成本 | $[X] |
| 当前股价 | $[X] |
| 行权价距现价 | X% |

---

## 💡 可以聊什么？

### 1. 这笔操作的核心逻辑
- 为什么选 [标的]？（结合近期新闻/走势）
- 为什么用 [策略] 而不是直接买股票？
- 这个行权价 / 到期日的选择逻辑是什么？
  - 距现价 X%，意味着……
  - 到期日距今 X 天/月/年，时间价值的考量是……

### 2. 可以科普的概念
- [策略名称] 是什么？一两句话概括：「……」
- 可以提：「不理解的朋友可以参考我之前关于 [概念] 的视频」
- 生活化类比建议：「……」

### 3. 数字亮点（让观众记住的关键数字）
- 净成本 / 净收入：$[X]
- 盈亏平衡点：$[X]
- 最大风险：$[X]
- 最大收益：[有限 $X / 无限]
- 年化回报率（如适用）：X%

### 4. 风险在哪？怎么处理？
- 最坏情况：如果 [标的] [跌破/涨过] $[X]……
- 应对方案建议：[Roll 展期 / 接受行权 / 止损]
- 保证金 / Margin 提醒（如适用）

### 5. 和大盘/系列目标的关系（可选）
- 这笔操作对百万之路账户的整体影响
- 和当前大盘环境的关系
```

## 策略识别指南

| 截图特征 | 策略判断 |
|---------|---------|
| Sell to Open + Put | Sell Put（卖看跌期权） |
| Sell to Open + Call（有持股） | Covered Call（备兑看涨） |
| Buy to Open + Call（到期 > 1年） | LEAPS Call |
| Buy Call + Sell Put，同行权价同到期日 | Synthetic Long（合成多头） |
| Buy LEAPS Call + Sell 近期 Call | PMCC（穷人版备兑） |
| Buy to Close | 平仓操作 |

---

## 组合策略处理

当截图包含多笔操作构成一个组合策略时（如合成多头）：

1. **先识别这是组合策略**，列出每条腿的参数
2. **计算净成本/净收入**：如 Buy Call $111.99 + Sell Put $81.99 = 净成本 $30.00/share = $3,000
3. **谈话要点额外增加**：
   - 为什么用组合而不是单腿？
   - 两条腿之间的关系是什么？
   - 保证金要求 vs 直接买股票的资金差异

---

## 规则

- **只给思路，不写台词** — 天哥自己在镜头前发挥，skill 不替他写话术
- **数字必须具体** — 不说"成本较低"，说"净成本 $3,000，相当于直接买股票的 X%"
- **必须包含风险点** — 每笔操作都要有风险提示，不能只讲好处
- **科普建议要实用** — 只推荐真正值得科普的概念，不是每笔操作都需要科普
- **提醒关联视频** — 如果某个概念之前视频讲过，建议天哥提一句「可以参考之前的视频」
