# 百万之路 — 本期视频大纲生成器

## Overview

通过逐步提问，收集本期视频所需的所有信息，填充到标准大纲模板中，输出一份完整的本期 outline。

**不写完整脚本。** 天哥会对着 outline 自己录音、自己写脚本。

---

## 执行流程

按以下顺序逐步提问，每次只问一组相关问题，等天哥回答后再问下一组。

### 第1轮提问：基本信息

```
这期是第几期？今天日期是？
账户当前总市值大概多少？（用于计算距离100万的进度）
```

### 第2轮提问：大盘情况

```
这周市场发生了什么大事？（1-2件，简单说就行）
SPX 这周涨跌多少？VIX 目前多少？
FedWatch 有什么变化吗？（没变化可以跳过）
```

### 第3轮提问：当天账户

```
今天账户涨了还是跌了？多少钱？
主要是哪些标的带动的？
有没有值得讲的浮盈/浮亏主角？
```

### 第4轮提问：本期操作

```
这期要讲哪些操作？（1-2个）
每个操作请提供：
- 标的 + 策略类型（Sell Put / LEAPS / Synthetic Long 等）
- 行权价 + 到期日 + 权利金/成本
- 为什么做这个操作？（一句话就行）

有截图的话可以直接发给我，我用 trade-analyzer 帮你分析。
```

### 第5轮提问：后市判断

```
最近有哪些大佬说了什么值得提的观点？（或者我帮你用 /market-brief 搜索）
你自己怎么看接下来的市场？操作方向是偏保守还是激进？
```

### 第6轮提问：补充

```
这期有没有什么特别想聊的？（科普点、诚实局、心态分享等）
HOOK 你想用哪种风格？（数字冲击 / 浮亏坦白 / 市场波动）
```

## 规则

- **逐步提问，不要一次性问完所有问题** — 每次只问一组，等回答后再问下一组
- **天哥说"跳过"或"没有"就跳过** — 不要强迫每个模块都有内容
- **不写完整脚本** — 只输出大纲，天哥自己录音发挥
- **大佬观点如果天哥没有提供，主动提议用 /market-brief 搜索**
- **操作分析如果天哥发了截图，主动用 trade-analyzer 分析**
- **数字必须来自天哥提供的信息，不得编造**
