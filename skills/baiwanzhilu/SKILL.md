# 百万之路 — 每周视频大纲生成器

## Overview

按「交易日志」十章模板逐章提问，收集本期视频所需的全部信息，输出一份可以直接拿来录制的视频大纲。

**不写完整脚本。** 天哥会对着 outline 自己录音发挥。

---

## 前置准备

每次启动前，按顺序读取：
1. `style_guide/tiange_style_guide.md`
2. `templates/weekly_1M_Journal_outline.md` — **本 skill 的核心模板**
3. `feedback/feedback_log.md`

---

## 执行流程

按模板十章顺序逐章提问。**每次只问一章**，等天哥回答后再问下一章。

每轮开始时先问：「这部分你有信息可以提供吗？还是我来帮你查？」

---

### 第1轮：基础信息

```
这期是第几期？日期？实盘第几周？
```

> 用于填写模板头部：集数 #、日期、周数。

---

### 第2轮：一、开场钩子 HOOK

```
这周最戏剧性或最有意思的事情是什么？
HOOK 风格你想用哪种？
  A) 数字冲击 — "这周我赚了 $X,XXX 权利金"
  B) 浮亏坦白 — "这周账户掉了 $XX,XXX"
  C) 市场波动 — "VIX 飙到了 XX，我做了这件事"
  D) 行权惊喜 — "XXX 被 assign 了，但我其实挺高兴"
  E) 自定义（你来说）
```

---

### 第3轮：二、市场背景 MARKET CONTEXT

```
这周市场数据你有吗？（SPY/QQQ/VIX/10Y 涨跌、重大事件）
有的话直接给我；没有的话我来查。
```

**如果天哥没有数据：** 自动执行完整市场调研（等同 `/market-brief` 级别），搜索并填入：
- SPY 本周涨跌 %
- QQQ 本周涨跌 %
- VIX 水平及状态
- 10年期美债收益率
- 本周显著事件（美联储、CPI、非农等）
- 对天哥交易的影响分析

> **规则：如果本周无重大事件，标注"市场平静，直接跳到投资组合"。**

---

### 第4轮：三、账户仪表盘 PORTFOLIO DASHBOARD

```
上周净值多少？本周净值多少？
资产分配明细：
  - 防御侧（QQQM / VOO / 个股）各多少？
  - 现金多少？
  - Wheel 策略多少？
  - LEAPS 多少？
（有仪表盘截图可以直接发给我）
```

**自动计算：**
- 本周涨跌金额和百分比
- 自第1天起总盈亏（基于 $400,000 初始资金）
- 百万目标进度百分比及差额
- 各类别实际占比 vs 目标占比偏差提示

---

### 第5轮：四+五、交易明细 + 专项情况

```
请上传本周的券商交易截图，我来帮你提取参数。
一共有几笔交易要讲？
```

**截图处理流程：**
1. 收到截图 → 调用 `/trade-analyzer` 提取每笔交易的参数
2. 展示提取结果，请天哥确认
3. 逐笔追问：
   - **为什么要开这一仓？**（一句话理由）
   - **如果行情走反了，你的计划是？**

**自动识别专项情况：**
- 如果交易包含**展期 (Roll)**：填写展期专项模板（原仓位→新仓位→净抵免额→展期次数→理由）
- 如果发生**指派行权 (Assigned)**：填写行权专项模板（有效成本价→备兑计划→目标离场价→90天时限）
- 如果买入**LEAPS**：填写 LEAPS 专项模板（RSI→Delta→展期计划→止损线）

**汇总本周权利金：**
- 卖出 Puts 总计
- 卖出 Calls 总计
- 展期净收益
- 总计收取 + 本周卖家 ROI

---

### 第6轮：六、本周教训 LESSON OF THE WEEK

```
这周最大的实操心得是什么？
要跟本周实际发生的交易挂钩，不讲抽象理论。
（如果没有特别的，可以跳过）
```

> 参考模板中的范例启发天哥：财报前持仓、展期成本、VIX 对权利金的影响等。

---

### 第7轮：七、计分板 SCOREBOARD

```
以下数据你有吗？（有仪表盘截图也可以直接发）
- 已坚持周数
- 总盈亏 P&L（金额 + 百分比）
- 累计收取权利金
- 平均每周权利金
- 胜率
- 预计达到 $1M 日期
```

---

### 第8轮：八、现金流瀑布 WATERFALL UPDATE

```
这个月有资金流动吗？
- 本月至今收取权利金
- 本月转入现金
- 本月定投至防御侧（QQQM / VOO 各多少）

没有的话这部分跳过。
```

---

### 第9轮：九、下周预演 + 催化剂

```
下周你有什么交易计划或观察名单吗？
比如：如果 XXX 跌到 $XX 我就卖 Put。
```

**自动调研（无论天哥是否提供）：**
- 下周经济日历（美联储会议、CPI、非农、PMI 等）
- 下周重要财报日程（盘前/盘后）
- 将调研结果填入模板，询问天哥对每个催化剂的应对计划

---

### 第10轮：十、CTA + 结尾

```
本期的互动话题想问观众什么？
比如："如果是你，你会怎么处理 [某个交易决策]？"
```

**自动填充固定内容：**
- 实盘周数 + 当前账户金额 + 本周涨跌
- 品牌 Slogan（$40万→$100万 真实历程）

## 规则

- **逐章提问，不要一次性问完** — 每次只问一章，等回答后再问下一章
- **每轮先问天哥有没有数据** — 有就用他的；没有就自行调研
- **天哥说"跳过"或"没有"就跳过** — 不强迫每个模块都有内容
- **不写完整脚本** — 只输出大纲，天哥自己录音发挥
- **截图优先** — 交易信息优先通过截图 + `/trade-analyzer` 提取，再补充理由
- **市场调研要完整** — 无数据时执行 `/market-brief` 级别的全面搜索（SPY/QQQ/VIX/10Y/经济日历/财报/专家观点）
- **数字必须来自天哥提供的信息或可靠调研，不得编造**
- **自动计算能算的都算** — 涨跌百分比、百万进度、年化回报、占比偏差等
